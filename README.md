# 🚀 Vue 3 + Vite + pnpm Project

一个使用最新技术栈构建的现代化Vue.js应用程序。

## ✨ 特性

- 🎯 **Vue 3.5.18** - 最新版本的Vue.js框架
- ⚡ **Vite 7.1.1** - 极速的构建工具和开发服务器
- 📦 **pnpm** - 高效的包管理器
- 🛠️ **TypeScript支持** - 类型安全的开发体验
- 🔧 **Vue DevTools** - 强大的调试工具

## 🛠️ 技术栈

| 技术 | 版本 | 描述 |
|------|------|------|
| Vue.js | 3.5.18 | 渐进式JavaScript框架 |
| Vite | 7.1.1 | 下一代前端构建工具 |
| pnpm | 9.12.3+ | 快速、节省磁盘空间的包管理器 |

## 📋 开发环境要求

- Node.js 20.19.0+ 或 22.12.0+
- pnpm 9.0+

## 🚀 快速开始

## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Compile and Minify for Production

```sh
pnpm build
```
